This demo video walks through the full workflow of the RAG Q&A application, from uploading a set of PDF documents to asking natural-language questions and receiving grounded answers with source citations. The recording covers the three main user journeys: document ingestion, interactive Q&A, and source verification.

The first segment shows the upload flow. A user drags several technical specification PDFs into the web interface, and the system processes them in real time, displaying a progress bar as documents are chunked and embedded. The second segment demonstrates the chat interface, where the user asks increasingly specific questions and the system retrieves relevant passages to construct its answers. Inline citation badges link each claim back to the original document and page number.

The final segment highlights edge cases: what happens when the knowledge base does not contain an answer (the system responds with a transparent "I don't know" rather than hallucinating), and how follow-up questions maintain conversational context. The video was recorded at 1080p using OBS Studio and edited in DaVinci Resolve with chapter markers for easy navigation.
