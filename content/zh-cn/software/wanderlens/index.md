WanderLens 是一款使用 React Native 和 TypeScript 构建的移动应用，帮助旅行者通过照片记录旅程。应用会自动按行程和地点整理照片，并将它们标注在交互式地图上，让你可以重温走过的路线。Firebase 负责身份验证、图片云存储以及跨设备实时同步。

每张照片在拍摄时会自动添加地理标签，并通过反向地理编码补充地名信息。用户可以创建行程相册、添加文字说明，并通过唯一链接与朋友分享精选照片。时间线视图按时间顺序串联所有照片，用户可以逐日浏览整个旅程。离线支持确保在网络覆盖有限的地区也能正常使用——照片会排队等待，在恢复连接后自动上传。

设计注重速度和简洁。图片在上传前会在客户端进行压缩以减少带宽消耗，缩略图通过 Firebase Cloud Functions 生成以加快图库加载速度。项目使用 Expo 进行构建和空中更新，方便在开发过程中快速迭代。
